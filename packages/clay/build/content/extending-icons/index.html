Extending Icons

Stacking Icons

Create custom icons with .icon-stacked, .icon-fg, and .icon-bg.

.icon-bg stands for icon background and should be declared first.

.icon-fg stands for icon foreground. It should be declared last. For several stacked icons in an .icon-stacked, the bottom most foreground icon should be declared right after .icon-bg and each subsequent foreground layer should precede the one before.
