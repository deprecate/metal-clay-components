Badges and Labels

Badges

Badges help highlight important information such as notifications or new and unread messages. Badges have circular borders by default, for rounded borders, See labels.

Default Primary Success Info Warning Danger 
<span class="badge badge-default">Default</span>
<span class="badge badge-primary">Primary</span>
<span class="badge badge-success">Success</span>
<span class="badge badge-info">Info</span>
<span class="badge badge-warning">Warning</span>
<span class="badge badge-danger">Danger</span>

Fixed Width and Height Badges (Not available in _styled)

Use badge-monospaced to create a badge that has a fixed width and height. It is useful for showing the number of new messages or pending requests in your application.
