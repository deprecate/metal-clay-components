Switches

Default Switches

Create css switches by adding class switch on <input type="checkbox">. <input type="checkbox"> must be declared before <label> and the label's for attribute must match the input's id attribute.

Use checked="checked" to turn a switch on by default.

<div class="form-group">
    <input checked="checked" class="switch" id="switch" type="checkbox" />
    <label for="switch">
        <span class="on">On</span><span class="off">Off</span>
    </label>
</div>

Contextual Switches

Use contextual classes to style switches.

<div class="form-group">
    <input checked="checked" class="edit switch switch-primary" id="contextual-switch0" type="checkbox" />
    <label for="contextual-switch0">...</label>

    <input checked="checked" class="edit switch switch-success" id="contextual-switch1" type="checkbox" />
    <label for="contextual-switch1">...</label>

    <input checked="checked" class="edit switch switch-info" id="contextual-switch2" type="checkbox" />
    <label for="contextual-switch2">...</label>

    <input checked="checked" class="edit switch switch-warning" id="contextual-switch3" type="checkbox" />
    <label for="contextual-switch3">...</label>

    <input checked="checked" class="edit switch switch-danger" id="contextual-switch4" type="checkbox" />
    <label for="contextual-switch4">...</label>
</div>

Disabled Switches

<div class="form-group">
    <input class="switch" disabled="disabled" id="switch" type="checkbox" />
    <label for="switch">
        <span class="on">On</span><span class="off">Off</span>
    </label>
</div>

Adding Icons

Add icons to the switch by adding the font-awesome class name to <input class="icon-pencil switch" type="checkbox">.

<div class="form-group">
    <input checked="checked" class="icon-pencil switch" id="switch" type="checkbox" />
    <label for="switch">Edit</label>
</div>

Alternative Mark Up for Switches

Having to supply a unique id for each switch can become tedious if you have a lot of switches. This alternative layout doesn't require a unique id, but requires you to use an extra element of mark up.

<div class="form-group">
    <label class="switch switch-primary">
        <input type="checkbox" />
        <span class="skin"></span>
        <span class="off">Off</span>
        <span class="on">On</span>
    </label>
</div>
