Vertically Centering Content

Use class flex-items-center to vertically center content. Content should be wrapped in an html element such as a div. You can horizontally center content by setting text-align: center;, margin-left: auto; margin-right: auto; or both on the content element and its children.

    <div class="flex-item-expand">...</div>

Flexbox Cards

Change the orientation of cards with minimal style changes.

<style>
    .card.flex-container {
        width: 300px;
    }

    .card.flex-container-horizontal {
        width: 50%;
    }

    .card.flex-container-horizontal .figure {
        width: 135px;
    }
</style>
