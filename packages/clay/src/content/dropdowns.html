Square Borders

Add class square-borders on an element to remove rounded borders. You can also add it to the parent to remove rounded borders on all children elements.

Scrolling Content

Add class inline-scroller to any element and set the height or width to make it scroll inline! The default height is 125px, but can be overridden to any height/width.

Mega Menu

Add class mega-menu to any dropdown to make the dropdown-menu BIG. The default width is 500px, but can be overridden to any width. The mega-menu-full class can be used to make the dropdown as wide as the navigation, see dropdown examples.
